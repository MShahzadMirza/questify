# Questify: Summer Tasks Progress App

This is a simple web-based application designed to help you track the progress of tasks in a RPG-style format. The app uses modern HTML, CSS, and JavaScript with local storage for data persistence.

## Features

- **RPG Elements**: Modern RPG-style graphics and animations to make the progress tracking experience visually appealing.
- **Mobile First Design**: Ensures a smooth user experience across various devices, including mobile phones.
- **Local Storage**: Stores tasks data locally in the browser, ensuring data persistence even after refreshing or closing the app.
- **Progress Tracking**: Allows you to track and visualize your task completion.

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/MShahzadMirza/questify.git
   cd questify
   ```

2. **Install Dependencies** (if any):
   - Ensure you have Node.js and npm installed.
   - Run `npm install` to install all dependencies.

3. **Run the Application**:
   ```bash
   npm start
   ```
   This will serve the app on [http://localhost:3000](http://localhost:3000).

## Usage

1. **Add Tasks**: Click the "Add Task" button to add new tasks with titles and descriptions.
2. **Mark Tasks as Completed**: Double-click or check the box next to a task to mark it as completed.
3. **View Progress**: The app will display the total number of tasks, completed tasks, and progress percentage.
