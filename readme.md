# 🌸 Questify Project Brief

## Project Name
Questify

## Main Idea
Questify is a cozy RPG-style study companion web app for students.

The goal is to make studying feel like a relaxing adventure instead of a boring task list.

The app should combine:
- Study planner
- Timetable generator
- Summer task manager
- Exam preparation assistant
- RPG progression system
- Cute animal companions

The theme should feel:
- Cute 🌸
- Cozy ☁️
- Lazy summer vibe 🍵
- Relaxing but motivating

# 🎯 Task System

Students create tasks inside subjects.

Example:

Mathematics:

- Complete Algebra chapter
- Practice questions
- Revise formulas


Tasks should:

- Be completed/uncompleted
- Give rewards
- Affect subject progress
- Appear in Today's Quests


Flow:

Create Task

↓

Today's Quest

↓

Complete Task

↓

Rewards + Growth

- Include breaks
