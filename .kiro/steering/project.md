# Questify Project Overview

## Project Name

Questify 🌸🦊

## Vision

Questify is a cozy RPG-style study companion web app designed to make studying feel like a relaxing adventure instead of a boring responsibility.

The feeling should be:

"I am not just studying.
I am growing my world with my companion."

## Main Concept

Questify combines:

- Study planner
- Task manager
- Timetable generator
- Exam preparation assistant
- Summer homework planner
- RPG progression system
- Cute companion system

The app should feel like:

- Cozy game
- Personal study assistant
- Friendly companion

## Main Character

Qi 🦊

Qi is the default study companion.

Personality:

- Calm
- Friendly
- Supportive
- Encouraging

Qi should:

- Motivate students
- React to progress
- Celebrate achievements
- Encourage breaks

Example messages:

"Small steps create big adventures 🌱"

"Great job! Let's continue together 🦊"

"Don't forget to rest too ☁️"

## Current Technology

Frontend only:

- HTML
- CSS
- JavaScript

Current structure:

index.html

style.css

script.js

The project intentionally uses simple technology because the developer is learning.
