# Questify Design System

## Design Vision

Questify should feel like:

- A cozy RPG game
- A relaxing study room
- A friendly summer adventure

The user should feel calm, motivated, and happy while using the app.

The main feeling:

"Study peacefully and grow slowly." 🌸

---

# Visual Style

Questify uses a:

- Cute style
- Cozy style
- Soft summer theme
- Pastel aesthetic

Avoid:

- Serious school dashboard designs
- Dark themes
- Harsh colors
- Corporate looking interfaces

---

# Color Direction

Main colors:

## Pink 🌸

Used for:

- Titles
- Buttons
- Highlights
- Rewards

Example:

Soft pink tones

---

## Cream / Beige ☁️

Used for:

- Backgrounds
- Cards
- Cozy spaces

---

## Green 🌱

Used for:

- Growth
- Plants
- Progress
- Nature elements

---

## Light Blue 🍃

Used for:

- Sky elements
- Calm backgrounds
- Cottage themes

---

# UI Style

All major sections should use:

- Rounded cards
- Soft shadows
- Smooth animations
- Comfortable spacing

Existing main style:

```css
.card
```

New sections should follow the same style.

---

# Cards

Cards represent different parts of the player's world.

Examples:

- Qi area
- Study Garden
- Profile
- Quests
- Timetable
- Exams

Cards should feel like objects inside a cozy room.

---

# Buttons

Buttons should be:

- Rounded
- Friendly
- Soft colored
- Easy to tap

Avoid sharp square buttons.

Buttons should feel like game actions.

Examples:

- Start Quest
- Complete Task
- Choose Companion
- Grow Plant

---

# Animations

Animations should be:

- Gentle
- Smooth
- Relaxing

Good examples:

- Floating
- Small movements
- Soft transitions
- Progress growth

Avoid:

- Fast flashing
- Aggressive animations

---

# Qi Companion Style

Qi 🦊 should always feel:

- Friendly
- Warm
- Supportive

Qi messages should encourage students.

Good:

"Great job! One more step 🌱"

"Remember to rest too ☁️"

Avoid:

"You failed"

"Work harder"

---

# RPG Feeling

The interface should make studying feel like progress in a game.

Use:

- XP
- Levels
- Coins
- Growth
- Friendship
- Achievements

Rewards should feel positive.

---

# Mobile Design

Questify should work well on phones.

Important:

- Buttons should be easy to tap
- Text should be readable
- Cards should fit smaller screens
- Layout should stay comfortable

---

# Design Rule

Every new feature should answer:

"Does this make studying feel more like a cozy adventure?"
